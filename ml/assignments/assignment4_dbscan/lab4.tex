\section*{Lab 4: DBSCAN}

\subsection*{Overview}
This lab follows Lecture 4 on DBSCAN. The goal is to implement density-based clustering from scratch with NumPy and to study how $\varepsilon$ and \text{MinPts} control the result.

Use the same notation as in the lecture:
\begin{itemize}
  \item $X = \{x_1,\dots,x_n\}$ is the dataset.
  \item $N_\varepsilon(x_i) = \{x_j : \|x_j - x_i\|_2 \le \varepsilon\}$ is the $\varepsilon$-neighborhood.
  \item A point is \emph{core} if $|N_\varepsilon(x_i)| \ge \text{MinPts}$.
  \item A point is \emph{border} if it is not core, but lies in the $\varepsilon$-neighborhood of a core point.
  \item A point is \emph{noise} if it is neither core nor border.
\end{itemize}

\subsection*{Policy}
\begin{itemize}

  \item Do not use a library DBSCAN implementation for the main task.
  \item Plotting libraries are allowed for visual inspection.

  \item Use \texttt{min\_pts} in code, which corresponds to \text{MinPts} in the lecture.
\end{itemize}
